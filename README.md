# 🚀 Liand Hub | Personal Portfolio & Feedback Ecosystem

Selamat datang di **Liand Hub**, pusat ekosistem digital dan portofolio interaktif milik [Liand Al Haq](https://liand-hub.vercel.app/). 

Proyek ini bukan sekadar portofolio statis biasa. Liand Hub dirancang dengan arsitektur **Frontend modern yang responsif** dan ditenagai oleh **Custom Backend (Node.js/MongoDB)** untuk menangani interaksi pengunjung secara *real-time*.

## ✨ Fitur Unggulan (Enterprise-Level)
- **Live Review System:** Pengunjung dapat memberikan rating (bintang) dan ulasan yang langsung tampil secara *real-time*.
- **Developer Response:** Sistem balasan admin terintegrasi layaknya ekosistem ulasan Play Store.
- **Smart Anti-Spam & Rate Limiter:** Dilindungi dengan sistem *Device ID* persisten dan batasan *request* untuk mencegah serangan spam.
- **Bug & Suggestion Report:** Pengunjung dapat mengirimkan *feedback* beserta bukti *screenshot* (Base64 Image Processing).
- **Advanced Security:** API Key Hashing (SHA-256) dan Enkripsi AES-256 untuk keamanan data tingkat tinggi.
- **Matrix Easter Egg:** Interaksi rahasia pada UI profil (Coba klik foto profil di web!).

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Variables, Dark Mode), Vanilla JavaScript.
- **Backend (Feedback Monitor):** Node.js, Express.js, Mongoose (MongoDB).
- **Deployment:** Vercel (Frontend) & Railway (Backend).
